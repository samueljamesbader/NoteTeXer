LaTeX Settings

Macros

You can put any LaTeX above to run it, and it will be run before rendering your notes, so this is a good place to define macros. You can also test the macros and the results will be shown below. Include the $$ around everything! See MathJax documentation for what will work (eg. you can't import arbitrary packages, sorry!)

Autocomplete

One command per line. If there's a space in the line, that's where the cursor will end after the command is autocompleted. So, for instance, a line braket{ }{ } would autocomplete the physicists' braket command, placing the cursor in between the (first) curly brackets.

Shortcuts

One shortcut per line. First the shortcut (eg. uufr), then a space, then the text which should replace it (eg. \frac{/#/}{/#/}). As above, the "anchor" (/#/) can be used to provide convenient navigation points. When the shortcut runs, the first anchor on the line will be selected, so that you may begin typing to replace it. Lines beginning with space are ignored.

What is this?

This is a tool for quickly taking notes with LaTeX, on both desktop and mobile devices.

Hi, I'm Sam Bader, a soon-to-be PhD student in Engineering Physics, and I designed Note$\TeX$er as a resource for students in the math and science community. I started this project because I couldn't find any free tools out there which would let me build my own LaTeX-based quick-references, to have with me anywhere, on any device.

That's really all this is, a simple note-taking app, with live LaTeX rendering built in. It includes an autocomplete and a user-configurable mobile-friendly array of shortcuts, so you can even type LaTeX quickly on your phone's native keyboard. In addition, there's a note-sharing feature so you can make selected formulas searchable to others and save them some effort! :)

Can I try?

Yes, literally right now!

If you want to try this out, go back to the home page and sign up (takes just a few seconds), and your account will automatically be populated by a series of notes that form a ten-minute tutorial on NoteTeXing efficiently.

The tool is provided free, as is, with no guarantees whatsoever. If you have suggestions, or would like to contribute to the project, follow the link above and talk to me! If this tool becomes popular, and requires more space on the Parse server, then I may have make it donation-supported, but in the meantime, it's available at no cost, because I just love you.

Acknowledgements

Many thanks to these projects!

The software makes use of MathJax to live-render your LaTeX, jQuery Mobile for the smooth front-end, Parse for the simple backend, and Jquery Text Complete to auto-complete your LaTeX. The licenses for each of these wonderful tools can be found on their respective websites.
